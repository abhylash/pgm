PROGRAM 9 – FROM ZERO TO WORKING (NO CONFUSION)
Folder Structure (inside full-stack)
program9/
 ├── employee-api/
 │   └── db.json
 └── employee-client/   (React App)

PART 1 – BACKEND (Mock REST API using json-server)
Step 1: Create Backend Folder
mkdir program9
cd program9
mkdir employee-api
cd employee-api

Step 2: Create db.json
nano db.json


Paste this:

{
  "employees": [
    { "id": 1, "firstName": "John", "lastName": "Doe", "email": "john.doe@example.com" },
    { "id": 2, "firstName": "Jane", "lastName": "Smith", "email": "jane.smith@example.com" }
  ]
}


Save: CTRL+O → Enter → CTRL+X

Step 3: Run json-server
npx json-server --watch db.json --port 5000


You should see:

Resources
http://localhost:5000/employees


Leave this terminal OPEN.

Test:

http://localhost:5000/employees

PART 2 – FRONTEND (React App)
Step 4: Create React App

Open new terminal:

cd ~/full-stack/program9
npx create-react-app employee-client
cd employee-client
npm install axios

Step 5: Replace App.js
nano src/App.js


Paste:

import React, { useEffect, useState } from "react";
import axios from "axios";

function App() {
  const [employees, setEmployees] = useState([]);

  useEffect(() => {
    axios.get("http://localhost:5000/employees")
      .then(res => setEmployees(res.data))
      .catch(err => console.log(err));
  }, []);

  return (
    <div>
      <h2>Employee Management System</h2>
      <table border="1">
        <tr>
          <th>ID</th><th>First Name</th><th>Last Name</th><th>Email</th>
        </tr>
        {employees.map(e => (
          <tr key={e.id}>
            <td>{e.id}</td><td>{e.firstName}</td><td>{e.lastName}</td><td>{e.email}</td>
          </tr>
        ))}
      </table>
    </div>
  );
}

export default App;

Step 6: Start React
npm start


Open:

http://localhost:3000


You will see the employee table fetched from REST API.
